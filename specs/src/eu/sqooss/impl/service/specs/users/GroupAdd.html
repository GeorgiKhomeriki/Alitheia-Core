Adding Groups to the System

In order to allow user access, and to manage their privileges easily, the system should allow to create new user groups.

The system can create empty groups and add them to its groups list.

Example

Given the system only having the initial group list (see this spec), when we add the new group "sqo-oss developers", then the group list becomes:

Group Name
new users
alitheia
sqo-oss developers

Also, this newly created group has no privilege.

Once this new group is added, when we try to add it again, the group list is not modified:

Group Name
new users
alitheia
sqo-oss developers
