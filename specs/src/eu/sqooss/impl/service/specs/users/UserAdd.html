Adding Users to the System

In order to allow user access, there is a need to create user accounts.

The system can creates users that are identified by name, email and password.

Example

Given the system only having the initial user list (see this spec), when we add the new user "test-user", having the email address "test-user@sqo-oss.org" and password "test-123. The default group is new_users_group" then the user list becomes:

User Name
alitheia
test-user

The new user by default is in the new_users_group group.

Trying to add the same test-user again is denied.
