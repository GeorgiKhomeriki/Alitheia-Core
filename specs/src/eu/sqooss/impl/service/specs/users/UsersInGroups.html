Moving Users in Groups

Since the system allows to create groups and users, it also allows to move users from one group to another.

Example

Given the system has the followin two groups:

Group
humongous
clones

And that the group "humongous" contains the following users:

User
ade
padams
sebas

While the clones" group is empty.

When we move the following users to the empty group:

User Name
padams
ade

Then, the groups composition will be the following:

Users in "humongous"
sebas
Users in "clones"
ade
padams
