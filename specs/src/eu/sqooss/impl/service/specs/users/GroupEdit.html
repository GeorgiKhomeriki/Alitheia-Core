Editing Groups of the System

Since the system allows to create groups, it also allows to change their description.

Example

Given the system only having the initial group list (see this spec), and an extra group named "humongous", containing the following users:

User
padams
sebas

When renaming the group to "humongous-ish", the users aren't removed, so the final user list for the group will be:

User Name
padams
sebas
