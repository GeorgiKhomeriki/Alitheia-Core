Removing Users from the System

In order to have control over the system, there should be a way to remove user accounts.

The system can delete users identified by their name.

Example

Given the system only having the initial user list (see this spec), when we add the new user test-user, having the email address test-user@sqo-oss.org and password test-123 the user list becomes:

User Name
alitheia
test-user

After the user was added, we remove it. The user list now contains only the initial user:

User Name
alitheia
