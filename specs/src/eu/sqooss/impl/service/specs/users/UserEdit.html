Editing Users of the System

Since the system allows to create user, it also allows to edit them. It is possible to edit the email and password of an user.

Example

Given the system only having the initial user list (see this spec), when we add the new user test-user, having the email address test-user@sqo-oss.org and password test-123, and using the default group new_users_group, the user list becomes:

User Name
alitheia
test-user

The user now has the following properties:

User Name Encoded Password Email
test-user 37eb19184eb9e4983307ec14d5441f731334fd473dee8556ed6bc7682fc4da32 test-user@sqo-oss.org

After changing the email address to test@sqo-oss.org, the user has the following properties:

User Name Encoded Password Email
test-user 37eb19184eb9e4983307ec14d5441f731334fd473dee8556ed6bc7682fc4da32 test@sqo-oss.org

After changing the password to new-password, the user has the following properties:

User Name Encoded Password Email
test-user b8b9f8f23992ebc2617febc03d92ecb1763fba7b77a5d053b69c416bad18a369 test@sqo-oss.org
