SQO-OSS Connector Library

The SQO-OSS connector library (SCL) provides a Java interface to the web-services of the Alitheia Core so that the UI can retrieve project and measurement data from the Core.

All the UI component will use this library to do their job. Through this library we can access the high-level features of the system. It allows to manage the users, projects and metrics.

Important Note

The specifications from this set assume the Alitheia Core system is running at the default location.
