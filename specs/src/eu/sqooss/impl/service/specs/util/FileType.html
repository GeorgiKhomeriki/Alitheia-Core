FileType Handling

The FileType class is a static utility class that maps file names and path names to (faux) MIME types, mostly by filename extension checking.

Example

When the system startups, it should be connected to the database.

The FileType methods use String.split a lot; when you split a string with just a single separator you get 2. And empty string yields 1
