Removing Projects from the System

Since the system allows to create projects, it also permits to delete them.

Example

Given the system has the following two projects:

First the project "iCopy" which source repository lies in "../specs/data/iCopy/svn".

Second the project "savonet" which mail archive lies in "../specs/data/savonet-mail/mail".

In other words, the project list is:

Project Name
iCopy
savonet

When we remove the project "iCopy", then the project list becomes:

Project Name
savonet
