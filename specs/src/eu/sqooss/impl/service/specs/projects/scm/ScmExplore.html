Exploring the Project Source History

In order to allow analysis involving the code history, the system should allow to explore projects source control management.

The system can create projects linked to a source control repository. Such a project will be available for source history exploration in order to allow the metrics to do their job.

Example

Given the system has a project named "iCopy" which source repository lies in the "../specs/data/iCopy/svn".

Then this project has a source code history.

Further Details
