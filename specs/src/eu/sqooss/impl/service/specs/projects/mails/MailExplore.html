Exploring the Project Mail History

In order to allow analysis involving the mail traffic within projects, the system should allow to explore projects mailing lists archives.

The system can create projects linked to a set of mailing list archives. Such a project will be available for mail traffic exploration in order to allow the metrics to do their job.

Example

Given the system has a project named "savonet" which mail archives lie in the "../specs/data/savonet-mail/mail".

Then this project has a set of mailing lists archives.
