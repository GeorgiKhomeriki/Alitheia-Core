Selected Metrics

Each metric plugin contains one or more metric. The metrics can be enabled or disabled for a specific project.

Example

The system comes with registered, but not installed metric plug-ins. After installing the "Wc metric plug-in", adding a "test-project" and synchronizing the metric plugin, we have the following metrics in the project:

Plugin Name Selected
LOC true
LOCOM true

Metrics can be selected or deselected. After deselecting the LOC metric, we have the following metrics in the project:

Plugin Name Selected
LOC false
LOCOM true

Selecting again the LOC metric, we have the following metrics in the project:

Plugin Name Selected
LOC false
LOCOM true
