Metric Uninstall

Quality is meassured with different metric plugins. In order to use a plug-in, it needs to be installed.

Example

The system comes with registered, but not installed metric plug-ins. The default plug-in list is:

Plugin Name Installed
Productivity metric plug-in false
CLMT plug-in false
Wc metric plug-in false

After installing the "Productivity metric plug-in", the list becomes:

Plugin Name Installed
Productivity metric plug-in true
CLMT plug-in false
Wc metric plug-in false

Uninstalling the "Productivity metric plug-in" gives back the original list:

Plugin Name Installed
Productivity metric plug-in false
CLMT plug-in false
Wc metric plug-in false
