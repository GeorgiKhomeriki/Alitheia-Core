The Wc (Word Count) plugin is reminiscent of the UNIX(tm) word count utility; it reads in a file as a sequence of text lines and counts the number of lines, non-blank lines, comment lines and total number of words in the file. The Wc plugin defines four metrics:

  • Wc.loc - total lines in the file
  • Wc.locom - lines of comments in the file
  • Wc.lonb - total non-blank lines in the file
  • Wc.words - total number of words in the file
